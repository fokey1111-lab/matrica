# Seasonality RSI Divergence Site

Статический сайт для загрузки пользовательских данных и расчёта алгоритма Seasonality RSI Divergence.

## Что умеет
- загрузка `.xlsx`, `.xls`, `.xlsm`, `.json`
- выбор листа, колонки даты и двух активов
- расчёт сигнала по логике Seasonality RSI Divergence
- графики цен, RS и RSI
- таблица месячных точек

## Формат Excel
Рекомендуемый лист: `Данные`
- колонка даты
- две числовые колонки активов

## Деплой
Загрузите файлы в корень репозитория и задеплойте на Vercel.
